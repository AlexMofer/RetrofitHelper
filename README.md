Retrofit Helper
===============

介绍
---

Retrofit辅助库，Gson解析，请求日志打印，解决回调强引用问题。

使用
---

**引用:**
```
dependencies {
    ...
    implementation 'io.github.alexmofer.retrofit:retrofit-helper:2.9.0.4'
    ...
}
```
