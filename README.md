Retrofit
========

<img src="icon.png" alt="Icon"/>

Retrofit辅助库

介绍
---

支持库，一些工具类及版本控制兼容类，一些特殊监听器。
- **CallFactory**

    Call对象生成工厂。
- **WeakCallback**

    弱引用化的回调。
- **CallbackWrapper**

    回调包装器。
- **SimplifiedCallback**

    精简化的回调。
- **TinyCallback**

    极简化的回调。

先决条件
----

minSdkVersion 9

入门
---

**引用:**

```java
dependencies {
    ...
    implementation 'am.util:retrofit:2.3.0'
    ...
}
```

注意
---

- 支持库中包含混淆文件，无需额外增加混淆指令
